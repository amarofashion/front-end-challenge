# To Get Started:
```
- Install node and make sure npm (Node Package Manager) is also installed.

```
# Dependencies:
-  Node v4 and NPM
- "express": "^4.13.3"
- "body-parser": "^1.14.1"
- "consign": "^0.1.2"


# Run the project:
```
- npm install (to install the dependencies from package.json).
- npm start
- Access: http://localhost:3000/
```
