# AMARO front-end challenge

An React Redux application

## Dependencies

- Node js (lts recommended) 
- NPM
- Webpack
$ sudo npm i -g webpack
- stylus
$ sudo npm i -g stylus

## Running

1. Install npm dependencies
$ npm i
2-a. Development
$ npm start | npm run start
2. Production
$ npm start prod | npm run start prod
