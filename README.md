Amaro Demo
---
O demo foi construido usando React/Redux com Webpack e SASS

### Instruções

```bash
npm install
npm start
```

E o build:
```bash
npm run build #into /dist
```

### Tests

Os unit tests usam Mocha/Expect. E Enzyme para os components

```bash
npm test
```

### Code style

ESLint com styleguide do Airbnb.
```bash
npm run lint
```
