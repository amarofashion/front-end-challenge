# Clone

```shell
git clone https://github.com/selique/front-end-challenge.git

```

# Prerequisites

In order to run this project node.js and npm both need to have been installed.

- NPM/Yarn
- Node.js


# Deployment

```
yarn build
```

## Development
```
yarn start
```

## Tests

```
yarn test
```

```
yarn eject
```
